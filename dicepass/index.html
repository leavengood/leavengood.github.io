Diceware Password Idea Generator

This page uses the EFF Large Wordlist and the built-in browser secure random number generator to create possible words to use for a password or passphrase. The dice roll used for each word is shown with the word. Everything is done within the browser, feel free to inspect the source code to be sure.

Take some of these words, these numbers, and these symbols to make a password. The more funny and memorable, the better. Randomly capitalize some words for increased security.
